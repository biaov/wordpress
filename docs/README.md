#

## 本地查看项目

### 克隆项目

```sh
git clone https://github.com/biaov/wordpress.git --depth=1
```

### 安装依赖

:::: code-group
::: code-group-item npm

```sh
# 安装依赖
npm i

# 启动项目
npm start

# 打包项目
npm run build
```

:::
::: code-group-item yarn

```sh
# 安装依赖
yarn

# 启动项目
yarn start

# 打包项目
yarn build
```

:::
::: code-group-item pnpm

```sh
# 安装依赖
pnpm i

# 启动项目
pnpm start

# 打包项目
pnpm build
```
