# Uni-App 离线打包 Android 端

## 前言

- 使用过原来云打包

| 对比     | 离线打包                   | 云打包               |
| -------- | -------------------------- | -------------------- |
| 打包速度 | 20秒 ~ 60秒                | 10分钟 ~ 30分钟      |
| 打包难度 | 一般（第一次需要配置环境） | 简单（无需配置环境） |

- 以下的 appid，应用名称也可通过 manifest.json 文件获取

## 第一步：下载 App 离线 SDK

- [App 离线 SDK 下载](https://nativesupport.dcloud.net.cn/AppDocs/download/android.html)，注意和 HBuilderX 版本一致
- 解压下载的 SDK 压缩包
- 目录结构

```md
├── Android-SDK@4.87.82540_20251128 ------------------------ 下载的 sdk
│ ├── HBuilder-HelloUniApp ------------------------------ 示例项目
│ ├── HBuilder-Integrate-AS ----------------------------- 集成到 Android Studio 项目
│ ├── readme -------------------------------------------- 说明文档
│ ├── SDK ----------------------------------------------- 离线 SDK
│ ├── UniPlugin-Hello-AS -------------------------------- 插件示例项目
│ ├── .DS_Store ----------------------------------------- Mac 电脑配置文件，Window 电脑可删除
│ ├── Feature-Android.xls ------------------------------- 功能说明
└ └── license.md ---------------------------------------- 协议文件
```

- 把 **HBuilder-Integrate-AS** 文件夹单独复制出来一份，避免后续操作影响到示例项目，其他文件暂时不用管

## 第二步：下载安装 Android Studio

- [Android Studio 官网](https://developer.android.google.cn/studio?hl=zh-cn)
- 安装下载的 **android-studio-panda1-patch1-windows.exe** 文件
- 点击 open 选择第一步复制出来的 **HBuilder-Integrate-AS** 文件夹

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/69bab9b970e440118a38c0c831ad0279.png#pic_center)

- 使用 HbuilderX 生成本地打包 APP 资源

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/069efde76431447c8d9d54679241024c.png#pic_center)

- 复制 `app\unpackage\resources\你的id文件` 到 `HBuilder-Integrate-AS\simpleDemo\src\main\assets\apps` 目录下
- 前面那个路径是 HbuilderX 控制台打印的路径

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/57563e365a4741909c168cd3e74448ef.png#pic_center)

## 第三步：生成并配置签名文件

- 登录[开发者中心](https://dev.dcloud.net.cn/)
- 在应用管理 -> 点击应用 -> 各平台信息 创建以及查看离线 AppKey

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/7f722ea1f8284a97989a5e6a10c8ea05.png#pic_center)

- 在应用管理 -> 点击应用 -> Android 云端证书 -> 下载证书
- 得到一个 \*.keystore 文件

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/68aa50544d104bdbac48784a935b9699.png#pic_center)

- 打开 `HBuilder-Integrate-AS/simpleDemo/src/main/assets/data/dcloud_control.xml` 文件填写 [AppId](https://dev.dcloud.net.cn/)
- 打开 `HBuilder-Integrate-AS/simpleDemo/src/main/res/values/strings.xml` 文件填写应用名称，通过 manifest.json 文件获取
- 打开 `HBuilder-Integrate-AS/simpleDemo/src/main/res/drawable` 目录下的图标和启动图，替换为你的应用图标和启动图
- 打开 `HBuilder-Integrate-AS/simpleDemo/src/main/AndroidManifest.xml` 文件填写 appkey，以及你在 manifest.json 文件中设置的 Android 权限

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/c5b48c6dfd154eceaa097ad191182be6.png#pic_center)
![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/1f74f7c666dd48f0a159a927d35d63db.png#pic_center)

- 打开 `HBuilder-Integrate-AS/simpleDemo/build.gradle` 文件填写打包信息

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/105e81c3b31c4ddb926ddff838dbd221.png#pic_center)
