## 本地查看项目

### 克隆项目

```sh
git clone https://github.com/biaov/wordpress.git --depth=1
```

### 安装依赖

::: code-group

```sh [npm]
# 安装依赖
npm i

# 启动项目
npm start

# 打包项目
npm run build
```

```sh [yarn]
# 安装依赖
yarn

# 启动项目
yarn start

# 打包项目
yarn build
```

```sh [pnpm]
# 安装依赖
pnpm i

# 启动项目
pnpm start

# 打包项目
pnpm build
```

:::
